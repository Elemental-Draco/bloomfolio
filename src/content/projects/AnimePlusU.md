## About Anime+U

Anime+U is a responsive, user-friendly web application that allows users to search for anime titles, view detailed information, and build a personalized watchlist. Designed as a polished front-end project for showcasing practical web development skills, Anime+U delivers a clean browsing experience powered by real data from the Jikan API.

## Core Features

### Anime Search & Discovery
- **Real-Time Search**: Fetch anime results instantly using the Jikan REST API  
- **Dynamic Rendering**: Card-based layout for displaying titles, images, and metadata  
- **Result Dialogs**: Open detailed pop-ups for descriptions, genres, and episode counts  

### Watchlist Functionality
- **Add to List**: Save anime to a personal watchlist from search results or dialogs  
- **Local Storage Persistence**: Watchlist remains saved between sessions  
- **Manage Your Picks**: Easily remove or update items  


### Smooth UI Interactions
- **Info Dialogs**: jQuery-powered modals with accessible keyboard support  
- **Loading & Error Handling**: Clear feedback for slow or failed API responses  
- **Responsive Layout**: Optimized for mobile, tablet, and desktop screens  

---

## Technical Architecture

### Frontend Stack
Anime+U is built with clean, lightweight front-end technologies:
- **HTML5 & CSS3**
- **JavaScript (ES6)**
- **jQuery** for DOM interactions and dialogs  
- **Jikan API** for anime data from MyAnimeList  

### Core Implementation Highlights
- **Fetch API** for asynchronous data retrieval  
- **Modular JavaScript** for search, rendering, dialogs, and watchlist logic  
- **LocalStorage API** for persistent data  
- **jQuery UI Patterns** for modals and interactions
